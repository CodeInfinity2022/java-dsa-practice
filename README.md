# Java DSA Practice

This repository contains my Java-based Data Structures and Algorithms practice for coding interviews and problem-solving improvement.

## Structure
Problems are organized by topic:
- Arrays
- Strings
- Linked List
- Stack
- Queue
- Binary Search
- Trees
- Graphs
- Dynamic Programming
- Greedy
- Math
- Design

## Problem Format
Each problem includes:
- Problem summary
- Pseudocode
- Approach
- Time complexity
- Space complexity

## Naming Convention
`LC###_ProblemName.java`

Example:
`LC001_TwoSum.java`

## Goal
Build a clean, revision-friendly Java DSA repository with consistent practice and readable solutions.
